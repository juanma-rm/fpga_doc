# fpga_doc

FPGA documentation and summaries repository.

## Tools

- **tools/split_pdf_by_chapter.py** — Python script to split PDFs by chapter/section using a Markdown Table of Contents. Usage: `python split_pdf_by_chapter.py <toc_file> [--pdf <pdf_path>] [--output <dir>]`

## Contents

- **vitis_hls_2025_2/** — Vitis HLS UG1399 v2025.2 documentation, including the original pdf, PDF extracts by chapter, and markdown summaries by chapter (LLM-generated).
