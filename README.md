# fpga_doc

FPGA documentation and summaries repository.

## Tools

- **tools/split_pdf_by_chapter.py** — Python script to split PDFs by chapter/section using a Markdown Table of Contents. Usage: `python split_pdf_by_chapter.py <toc_file> [--pdf <pdf_path>] [--output <dir>]`

## Contents

- **vitis_hls_2025_2/** — Vitis HLS UG1399 v2025.2 documentation, including the original pdf, PDF extracts by chapter, and markdown summaries by chapter (LLM-generated).
- **vitis_ai_2025_2/** — Vitis AI documentation with PDF and TOC.
- **vitis_embedded_2025_2/** — Vitis Embedded documentation with PDF and TOC.
- **vitis_reference_guide_2025_2/** — Vitis Reference Guide documentation with PDF and TOC.
- **vivado_design_flows_2025_2/** — Vivado Design Flows documentation with PDF and TOC.
- **vivado_getting_started_2025_2/** — Vivado Getting Started documentation with PDF and TOC.
- **vivado_ip_2025_2/** — Vivado IP documentation with PDF and TOC.
- **vivado_ultrafast_design_methodology_2025_2/** — Vivado Design Methodology documentation with PDF and TOC.
