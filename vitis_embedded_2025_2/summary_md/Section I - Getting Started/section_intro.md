# Section I — Getting Started with Vitis

This section provides a brief overview of the AMD Vitis™ unified software platform and describes the installation requirements and procedures to install and run the tool.

> **Note:** `$XILINX_VITIS` indicates the AMD Vitis™ installation directory. It is set up after running:
> ```bash
> source <VITIS_INSTALL_DIR>/<VERSION>/settings.sh
> ```
