# Chapter 1 — Navigating Content by Design Process

AMD documentation is organized by **design processes** accessible on the Design Hubs page and the Design Flow Assistant. For the **Embedded Software Development** process, the relevant topics are:

| Topic | Description |
|-------|-------------|
| Creating a Platform Component from XSA | Import hardware from Vivado |
| Customizing a Pre-Built Platform | Modify existing platform BSPs/domains |
| Creating an Application Component | Create embedded apps from templates or examples |
| Run, Debug, and Optimize | Launch, debug, and profile on hardware |

> For other design processes (acceleration, HLS), refer to **UG1416** — Vitis Unified Software Platform Documentation Landing Page.
