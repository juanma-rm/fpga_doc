## Finding Additional Documentation

### Technical Information Portal

The AMD Technical Information Portal provides robust search and navigation for documentation: [https://docs.amd.com](https://docs.amd.com)

### Documentation Navigator (DocNav)

An installed tool for accessing AMD documents, videos, and support resources:

| Launch Method | Command |
|---|---|
| From Vivado IDE | Help → Documentation and Tutorials |
| Windows | Start → Xilinx Design Tools → DocNav |
| Linux | `docnav` |

> See *Documentation Navigator User Guide (UG968)* for more information.

### Design Hubs

Provide links to documentation organized by design tasks:

- In DocNav: click the **Design Hubs View** tab
- Online: visit the Design Hubs web page
