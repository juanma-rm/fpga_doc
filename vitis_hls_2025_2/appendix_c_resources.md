## Finding Additional Documentation

### AMD Technical Information Portal

Online search and navigation for all AMD documentation:
- URL: https://docs.amd.com

### Documentation Navigator (DocNav)

Installed tool for offline access to AMD Adaptive Computing documents, videos, and support resources.

**How to open DocNav:**
- From AMD Vivado IDE: **Help → Documentation and Tutorials**
- Windows Start menu: **AMD Adaptive SoC and FPGA Tools → DocNav**
- Linux command line: `docnav`

### Design Hubs

Links to documentation organized by design task and topic.

**How to access:**
- In DocNav: click the **Design Hubs View** tab
- Web: AMD Design Hubs web page
